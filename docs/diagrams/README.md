## Design Guidelines

**Colors:**
- Primary Blue: `#3B82F6`
- Success Green: `#10B981`
- Warning Orange: `#F59E0B`
- Purple (AI): `#8B5CF6`
- Gray: `#6B7280`

**Fonts:**
- Body text: Inter or Roboto
- Code: Fira Code or JetBrains Mono

**Icons:**
- Style: Lucide or Heroicons
- Size: Consistent across diagrams

## Usage

Add diagrams to documentation with:

```markdown
![Description](docs/diagrams/filename.png)
```

## Contributing

When adding new diagrams:
1. Follow the design guidelines above
2. Export at 2x resolution for crispy display
3. Save as both PNG and SVG (if possible)
4. Update the checklist above
5. Add alt text when referencing in docs
